Se hicieron los siguientes supuestos:
-La URL por defecto para generar una conexión es http://localhost:8080
-Sólo se ingresará una sesión en el servidor y no intentará ingresarse otra hasta que se reinicie el servidor.

Instrucciones:
1.-Abrir el proyecto con IntellijIdea.
2.-Presionar play (botón verde de Play es la esquina superior derecha).
3.-En caso de no poder hacerse el paso 2, presionar el mismo botón, pero situado al lado de la clase "MiniServerSocketExample" que está en en archivo con el mismo nombre.
4.-Una vez corriendo el servidor, ingresar al navegador y comenzar a ingresar las URL.
